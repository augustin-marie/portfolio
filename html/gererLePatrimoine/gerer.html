Application All You Need

J'ai utilisé git et git hub pour la gestion de version du PoC créé pour All You Need.
Git est une application de gestion de version décentralisée. Le dépôt était publique, mais cela avait été jugé peut gênant étant donné qu'il ne s'agissait que d'un proof of concept et pas d'une application en soi.

Réalisation en milieu professionnel en cours de seconde année

Utilisation d'un dépôt svn sur redmine

Pour le versionning et la sauvegarde du code, je suis passé par un dépôt svn héberger sur redmine.
Redmine est une application de gestion de projet et de sauvegarde de code. Elle est auto hébergée ce qui veut dire que le serveur sur lequel était hébergé le code appartenais à l'entreprise ADTi.
SVN (pour apache subversion) est un logiciel de gestion de version centralisée (contrairement à git).
